# Content Rules

## Fact Boundary

- Treat raw user material as the only source of truth.
- Keep a mental ledger of provided facts versus rewritten expression.
- Never invent: company, job title, school, degree, date, certificate, award, ranking, platform access, sales number, traffic number, conversion result, or tool proficiency.
- If the user implies but does not state a fact, either ask or write a conservative version.

## Language Style

Write concise Chinese resume language. Avoid empty claims such as `具有较强沟通能力`, `认真负责`, `学习能力强` unless tied to an experience.

Good bullet pattern:

`动作 + 对象 + 工具/方法 + 产出`

Examples:

- `整理 Amazon 竞品 Listing 信息，按价格、评分、卖点、关键词建立对比表，辅助选品和内容优化。`
- `使用 Excel 维护 ASIN、BSR、价格和评价数据，输出周度变化记录，支持运营复盘。`
- `协助撰写商品标题、五点描述和关键词，优化 Listing 信息完整度和搜索匹配度。`

When no metric is provided, use grounded verbs:

- `整理`, `收集`, `维护`, `协助`, `跟进`, `分析`, `复盘`, `输出`, `归档`, `核对`, `优化`, `梳理`

Avoid unsupported exaggeration:

- `主导`, `全面负责`, `精通`, `显著提升`, `大幅增长`, `深度参与`, `成功打造`, `行业领先`

## Section Strategy

Default sections:

1. `教育背景`
2. `项目经历`
3. `实习经历`
4. `技能证书`

Reorder only when the target job makes another order clearly stronger. For internships, education usually stays near the top.

## HR 10-Second Check

The upper half of the first page should reveal:

- Name and target role.
- School, major, degree, and graduation year.
- Two to five target-role keywords.
- The strongest relevant project or internship evidence.

If these are buried, reorder or compress.
