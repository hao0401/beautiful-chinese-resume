# Layout Rules

## Page

- A4 portrait.
- One page unless the user explicitly requests otherwise.
- Margins: 1.3-1.6 cm.
- Use whitespace and hierarchy instead of dense grids.
- Avoid decorative templates, heavy borders, dark backgrounds, large icons, and photo-heavy designs unless the user explicitly asks.

## Typography

- Chinese font: Microsoft YaHei, DengXian, SimSun fallback.
- Name: 18-22 pt, bold.
- Target role and contact: 9-11 pt.
- Section title: 10.5-11.5 pt, bold, restrained accent color.
- Body: 9.5-10.5 pt.
- Dates: same size as body, muted gray.

## Visual Style

Default style should feel clean, modern, and suitable for campus/internship applications:

- One restrained accent color such as teal, blue-gray, or deep green.
- Thin light dividers under section titles.
- No dense visible tables.
- Hidden tables may be used only to align headings and dates.
- Keep bullets short; prefer one-line or two-line bullets.

## Compression Order

When content overflows:

1. Remove weak self-evaluation or generic statements.
2. Merge repetitive bullets.
3. Shorten low-relevance details.
4. Remove low-relevance campus/award content.
5. Slightly reduce paragraph spacing.
6. Reduce body font only within readable limits.

Do not solve overflow by making the page look cramped.
